# 1.8 查询商家充值结果

#### 基本信息
```
api：dianwoda.seller.recharge-result.query

接口描述：商家成功调用充值接口，主动调用此接口查询充值结果。

```

#### 请求参数
字段名 | 类型 | 是否必须 | 示例 | 描述
---|---|---|---|---
seller_id|String|是|123456|商家编号，该参数必传
biz_no|String|是|5454yndd-45333-222|点我达充值业务流水号，建议接入方保存该值，为后续核账使用

```javascript
请求示例：
{
  "sellers": [
    {
        "seller_id": "123456",
        "biz_no": "5454yndd-45333-222"
    },
    ...
  ]
}
```

#### 响应参数
详见平台参数；

```javascript
正确响应结果示例：
{
	"code":"success",
	"message":"成功"
}
```
