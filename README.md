2.5 获取订单信息 |什么是恶劣天气加价？|点我达根据事先与接入方约定的加价规则，在出现恶劣天气时，由点我达平台主动给订单增加天气补贴的手段，使运力倾斜；默认禁用，若要开启可联系运营。
